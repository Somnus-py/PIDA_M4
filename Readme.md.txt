## Contexto
Este proyecto esta basado en el analisis de datos de una empresa farmaceutica interesada en expandir sus actividades en latinoamerica
Para lo cual a seleccionado 6 paises que serian los candidatos para dicha accion
Se nos ha solicitado realizar un estudio de analisis de datos de estos 6 paises utilizando una base de datos que nos han proporcionado con el fin de descubrir insght importantes para la toma de decisiones


## Tecnologías Usadas
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
- Power BI 
- Git 
- GitHub

## Conclusiones Más Importantes
1) Los datos indican la gran preponderancia de Brasil en las actividades
2) La falta de datos relevante, como casos de recuperados han dejado a Argentina fuera de la consideracion al considerse arriesgado por no tener inormacion suficuente
3) Recomendamos considerar la expansion de las operaciones a Brasil por el volumen de las operaciones
